# Easyfest — Identité visuelle v4 fmono (canonical)

Pack officiel validé par Pamela Giordanengo (présidente ZIK / Roots du Lac), 2 mai 2026.
Les versions antérieures (v1, v2, v3) sont archivées dans `_archived-*/`. **Ne plus les utiliser**.

## Concept

F monogramme stylisé dans un squircle coral premium. Le point d'accent ambre symbolise
le déliant — l'étincelle festive qui fait basculer un événement bricolé en festival pro.

## Fichiers

| Fichier | Usage |
|---|---|
| `app-icon.svg` | Icône app iOS/Android (1024×1024, squircle radius 224) |
| `emblem.svg` | Emblème seul, sans wordmark — favicon haute densité, watermark |
| `favicon.svg` | Favicon navigateur (32×32, squircle radius 7) |
| `logo-horizontal.svg` | Header web, signature email, document |
| `logo-vertical.svg` | Poster, splash screen, cover sociale |
| `logo-dark.svg` | Variante fond sombre (cream-on-coral inversée) |
| `mono.svg` | Tampon, embossage, impression mono |
| `og-image.svg` | Open Graph 1200×630 partage social |

## Typographie

- **Wordmark / display** : Source Serif 4 (ex. Source Serif Pro) Black 900, tracking -0.5 à -2
- **Body** : Inter 400/500/600/700
- **Mono** : JetBrains Mono

## Tagline

> **LE FESTIVAL PRO, SANS LE PRIX PRO**
>
> Inter Semibold, tracking 2.5px, uppercase, opacité 0.6 sur cream / 0.7 sur dark.

## Marges & tailles minimales

- Squircle isolé : marge minimale = ½ hauteur du squircle
- Logo horizontal : taille mini 80px de large (mobile), 120px (desktop)
- Favicon : ne jamais descendre sous 16×16
- Sur photo : utiliser `logo-dark.svg` ou `mono.svg` jamais le coral pur

## Interdits

1. Ne pas étirer, biseauter, ombrer ou ajouter de gradient au squircle
2. Ne pas séparer le point ambre du F
3. Ne pas utiliser sur fond visuellement chargé sans plaque cream
4. Ne pas remplacer la typo Source Serif par autre chose
